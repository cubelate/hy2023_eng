投资分析师（多赛道）
岗位职责

1.面向多赛道领域行业进行信息搜集和研究，并撰写行业和赛道分析报告；

2. 对所专注领域的投资项目与创始人进行沟通分析和机会挖掘；

3. 对行业内头部企业、智库、研究院、投资人等进行调研，跟踪重点公司动态；

4. 协助组织面向被投资企业提供管理赋能和增值服务。

任职要求

1. 理工科学历背景，硕士及以上学历，有知名VC/PE机构或产业头部企业实习经验优先；

2. 对科技赛道有持续的关注与热爱，特别在科技发展、数字化、新能源等领域有一定的研究，拥有相关产业背景或专业知识背景优先考虑；

3. 具备较强的专业分析能力、市场/行业研究能力和判断能力，能根据行业信息输入，形成自己独立的观点和看法；

4. 具有较强的逻辑思维能力和学习能力，良好的沟通能力，良好的自我驱动及高效执行。

投递简历
企业学习发展专家
岗位职责

1. 负责企业内部及投后企业的训战课程设计及交付，定期了解投后企业诉求调整课程安排；

2. 负责训战交付期的全流程支持，包括但不限于内容准备、讲师对接、会务安排、学院跟进等工作；

3. 做事积极主动，细心，亲和力强，善于调动资源实现复杂问题的解决及落地；

4. 对人才发展及学习成长有热情、有兴趣，有良好的人际关系理解力与链接能力；

5. 具有良好的音视频编辑技能及文案策划能力，熟练掌握PowerPoint、音视频编辑等软件；

6. 有志向成长为企业独立讲师/战略发展专家，对企业战略运营有较高兴趣。

任职要求

1. 985/211本科及以上学历的应届毕业生或毕业2年内的社招人员；

2. 具有企业内训、课程设计与交付、独立讲师等相关经验优先；

3. 有良好的英语读写及表达能力；

4. 对科技、投资、人力资源管理等领域有较高兴趣。

基金运营
岗位职责

4. 具备较强的数据和文书处理能力及优秀的沟通协调能力。
